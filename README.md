# Agent-Memory (v6)

一体化个人 AI 记忆系统。单文件夹复制即用，无需配置平台入口文件。

---

## 快速上手指南

### 它能做什么

记忆系统 = 让每个新会话都认识你的"外脑"。

- 日记与早晚规划（道痕六层 + 石头追踪）
- 周报与月度校准（置信度晋升管线）
- 四层记忆体系（L1-L4）+ AI 行为规则自动进化
- 记忆琥珀快照（改错可回滚）
- 深度推理协议（复杂问题先想清楚再动手）
- 记忆冲突检测（言行不一致时自动提醒）

### 四层架构

| 层级 | 名称 | 存什么 | 更新频率 |
|:----:|------|--------|---------|
| L1 | 核心层 | 身份、价值观、底线 | 每年 1 次 |
| L2 | 认知层 | 思维模式、AI 交互偏好 | 每 3 个月 |
| L3 | 行为层 | 工作节奏、石头追踪 | 每 3 个月 |
| L4 | 情境层 | 当前目标、卡点 | 每月 |

## 安装

```bash
npx skills add easonlao/memory-system
```

或直接复制 `skills/memory-system/` 目录到你的 AI 平台的 skills 目录。

然后对 AI 说 **"使用记忆系统"**，AI 会引导你完成：
1. 选择数据存放路径（`$MEMORY_DATA`）
2. 回答 33 题问卷（约 30 分钟）
3. 自动生成记忆文件和洞察报告

完成后说 **"检查记忆"** 确认系统状态正常。

## 信号词（日常使用）

| 你说 | AI 做什么 |
|------|----------|
| **"MS-HELP"** | **⭐ 统一入口：说任何需求，AI 自动匹配工作流** |
| "早上好" / "开始规划" | 早晨规划（读 L3/L4 + 定今日目标） |
| "今日复盘" / "写日记" | 晚间复盘（道痕六层 + 石头追踪） |
| "写周报" / "周复盘" | 周报整理 |
| "月度校准" | 月度校准（L3→L2→L1 晋升管线） |
| "检查记忆" | 健康检查 |
| "升级记忆系统" | 升级（只换系统文件，不动数据） |
| "回滚 xxx" | 从记忆琥珀恢复历史版本 |

> 每个工作流执行完毕后会自动推荐下一步，你不需要记住所有信号词。**只需记住 "MS-HELP" 一个入口即可。**

### 安全词（特殊模式）

| 安全词 | 效果 |
|--------|------|
| `闭眼执行` | 跳过深度推理，直接执行 |
| `忽略原则` | 跳过记忆冲突检测 |
| `安静模式` | 极致精简输出 |

## FAQ

### Q1：说"检查记忆"没反应？
1. 确认 config.json 存在（`{skill_dir}/config.json`）
2. 确认 `$MEMORY_DATA` 路径有效
3. 重新说"使用记忆系统"

### Q2：改错文件怎么办？
说"回滚 [文件名]"，从记忆琥珀恢复历史版本。

### Q3：换了电脑怎么办？
复制 `$MEMORY_DATA` 目录到新电脑，重新安装记忆系统即可。

### Q4：每天必须写日记吗？
强烈推荐。日记是后续周报/月报的数据基础。不写日记 = 系统白搭。

### Q5：问卷答案可以改吗？
可以。直接编辑对应文件，或说"更新核心层信息"等。

## 特点

- **一个技能搞定全系统** — memory-system 自带所有依赖
- **你掌控数据** — `$MEMORY_DATA` 目录是唯一需要备份的，迁移时复制即可
- **跨平台** — Claude Code、Cursor、Gemini CLI、GitHub Copilot 等
- **自动维护** — AI 在后台管理工作区日志、行为提炼
- **升级不影响数据** — 替换技能目录后说"升级记忆系统"即可
- **运行时帮助** — 说"MS-HELP"随时了解系统功能和工作流

## 从 v5.x 升级

备份 `$MEMORY_DATA` 目录后，对 AI 说 **"升级记忆系统"**，按指引操作即可。
