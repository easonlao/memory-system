
# 07 — 用户问卷（Q1-Q33）

&gt; 初始化时逐一回答全部 33 题。
&gt; 安装完成后作为题库池使用，系统根据记忆状态动态出题追问。
&gt; 答案写入对应记忆层。

---

## 出题逻辑（安装后使用）

| 优先级 | 条件 | 动作 |
|--------|------|------|
| P0 矛盾 | L1↔L3 审计发现差异 | 生成针对性追问 |
| P1 空白 | 某维度无数据 | 挑对应题目 |
| P2 低置信 | GL2 经验置信度 &lt; 0.5 | 轻量追问 |
| P3 刷新 | 到维护节点 | 触发对应层轻量刷新 |

每次最多 3 道，按优先级排序。
