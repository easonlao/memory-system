# 工作流 F：周报整理

触发："写周报" / "周复盘"

1. 确定本周覆盖的日期范围
2. 如果跨月，读取两个月份目录的日记
3. 提取 SSOT 汇总：完成事项、时间分配、能量状态
4. 提取**重复石头列表**（跨天出现的同一本质主石头）
5. 对比上周计划与实际完成
6. 输出周报摘要给用户确认
7. 用户确认后写入周记（`$MEMORY_DATA/USER/diary/YYYY/MM/YYYY-Www.md`，放在天数较多的月份目录）
8. **石头回顾**：汇总本周重复出现的主石头，输出总结："这周'{石头}'重复出现了 N 次，你想到了什么？"
   - 不晋升、不自动写入 L3
   - 目的是让用户反思，不是让系统记录
9. 按 SKILL.md 日记模板输出

末尾追加标记：`[🧠 Agent-Memory: 周报整理]` + `[🧠 路由: L3]`

执行完毕后，按 SKILL.md 通用推荐规则推荐下一步。
