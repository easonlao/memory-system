# Agent-Memory v5.0

## 安装

1. 将 `技能配置/唤醒记忆系统` 安装到平台的 skills 目录
2. 对 AI 说 **"安装记忆系统"**

安装向导会自动：配置入口文件 → 安装其他技能 → 创建记忆数据目录 → 引导完成问卷。

## 后续

- `$MEMORY_DATA` 是唯一需要备份的目录，迁移时复制即可
- 日常使用 AI 会自动管理记忆

## 文件说明

| 文件 | 用途 |
|------|------|
| `技能配置/唤醒记忆系统/` | **唯一的安装 Skill**（含依赖） |
| `技能配置/唤醒记忆系统/references/` | 其他技能、问卷、报告模板 |
| `Agent-Memory/` | 框架文档参考 |
