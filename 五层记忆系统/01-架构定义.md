# 01 — 架构定义

> 定义五层记忆系统的核心结构：每一层的含义、存储位置、数据流向。

---

## 五层架构

| 层级 | 名称 | 内容 | 变化速度 | 存储位置 |
|------|------|------|---------|---------|
| L1 | 元信息层 | 价值观、底线、核心驱动力、金句 | 年/永久 | `$MEMORY_ROOT/USER/L1-核心.md` |
| L2 | 认知模式 | 决策习惯、思维方式、交互偏好 | 季度 | `$MEMORY_ROOT/USER/L2-L3-规则.md` |
| L3 | 行为规律 | 能量周期、工作节奏、习惯 | 月度 | `$MEMORY_ROOT/USER/L2-L3-规则.md` |
| L4 | 情境状态 | 当前目标、卡点、项目进度 | 天/周 | `$WORKSPACE/.memory/MEMORY.md` |

### L1 元信息层

**定义**：你最底层的价值观、底线、核心驱动力。你"是什么样的人"的本质。

**特点**：
- 变化极慢，写一次管很久
- 必须有你确认才能写入或修改
- 不衰减，永久有效直到你主动改变

**来源**：
- 问卷 Q1-Q7
- L2 中持续 6 个月以上稳定的模式（AI 提案 + 你确认）

**文件**：`$MEMORY_ROOT/USER/L1-核心.md`

### L2 认知模式层

**定义**：你习惯怎么思考、怎么做决策、怎么接收信息。

**特点**：
- 变化较慢（季度级）
- 置信度管理，自动写入但需佐证
- 6 个月未观察到开始衰减

**来源**：
- 问卷 Q8-Q17
- 从 L3 行为规律归纳（AI 自动 + 置信度评分）

**文件**：`$MEMORY_ROOT/USER/L2-L3-规则.md`

### L3 行为规律层

**定义**：从项目级数据中提取的跨项目共性行为模式。

**特点**：
- 变化中等（月级）
- 从 L4 原始数据自动提取，不需要你确认
- 数据不出现后每周降置信度

**来源**：
- 多个项目的 L4 日记/进度数据泛化
- 对话中的行为观察

**文件**：`$MEMORY_ROOT/USER/L2-L3-规则.md`（与 L2 同文件但分章节）

### L4 情境状态层

**定义**：你现在在做什么，当前项目的状态、目标和进度。

**特点**：
- 变化最快（天/周级）
- 按项目隔离，不进入公共区
- AI 自动记录，不需要提炼

**来源**：
- 日常对话
- AI 自动记录日志/进度

**文件**：`$WORKSPACE/.memory/MEMORY.md`

## 数据流向

### 提炼方向（从具体到本质）

```
L4 项目数据（日志/进度/任务）
  │ AI 在月度校准中提取跨项目共性
  ▼
L3 行为规律（"截止日前你会冲刺"）
  │ 置信度 ≥ 0.7 → 考虑晋升 L2
  ▼
L2 认知模式（"你习惯先分析再行动"）
  │ 置信度 ≥ 0.9 持续 3 个月 → AI 提案
  ▼
L1 元信息（"结果比过程重要"）
  │ 你确认后锁定
```

### 预测方向（从本质到具体）

```
L1（"结果比过程重要"）
  │
L2（"截止日前会熬夜赶工"）
  │
L3（"最后 48 小时效率最高"）
  │
L4（"项目 A 还剩 3 天截止"）
```

## 系统常量

### 公共文件夹路径

| 常量名 | 路径 | 说明 |
|--------|------|------|
| `PATH_USER_L1` | `$MEMORY_ROOT/USER/L1-核心.md` | L1 核心层 |
| `PATH_USER_L23` | `$MEMORY_ROOT/USER/L2-L3-规则.md` | L2+L3 规则 |
| `PATH_AGENTS_GL1` | `$MEMORY_ROOT/AGENTS/GL1-铁律.md` | GL1 铁律 |
| `PATH_AGENTS_GL2` | `$MEMORY_ROOT/AGENTS/GL2-模式候选.md` | GL2 模式候选 |
| `PATH_AGENTS_PROJECTS` | `$MEMORY_ROOT/AGENTS/项目总览.md` | 项目总览 |
| `PATH_SKILLS_DIR` | `$MEMORY_ROOT/技能配置/` | Skill 模板源 |

### 工作区路径

| 常量名 | 路径 | 说明 |
|--------|------|------|
| `PATH_MEMORY_DIR` | `$WORKSPACE/.memory/` | 项目级工作记忆目录 |
| `PATH_MEMORY` | `$WORKSPACE/.memory/MEMORY.md` | L4 情境层 |
| `PATH_DIARY` | `$WORKSPACE/.memory/YYYY-MM-DD.md` | 工作区日志 |
| `PATH_GL2B` | `$WORKSPACE/.memory/.rules/` | GL2b 项目规则 |

### 目录结构

```
$MEMORY_ROOT/
├── USER/                          ← 你的画像（L1-L3）
│   ├── L1-核心.md                 ← 价值观、底线、驱动力
│   └── L2-L3-规则.md              ← 认知模式 + 行为规律
│
├── AGENTS/                        ← AI 侧数据
│   ├── GL1-铁律.md                ← 编译后的行为底线
│   ├── GL2-模式候选.md            ← 观察中的合作规律
│   └── 项目总览.md                ← 跨工作区索引
│
├── 五层记忆系统/                  ← 框架定义
│   ├── README.md
│   ├── 01-架构定义.md
│   ├── ...
│
├── 技能配置/                      ← Skill 模板
│   ├── 记忆助理.md
│   └── ...
│
└── CLAUDE.md

{工作区}/.memory/                  ← 项目级，按项目隔离
├── MEMORY.md                      ← L4 情境层
├── YYYY-MM-DD.md                  ← 工作区日志
└── .rules/                        ← GL2b 项目规则
```

## AI 侧结构

| 组件 | 内容 | 存储位置 | 加载方式 |
|------|------|---------|---------|
| GL1 铁律 | 从你 L1 编译 + 跨项目晋升 | `$MEMORY_ROOT/AGENTS/GL1-铁律.md` | 每次对话必读 |
| GL2a 场景规则 | 跨项目合作规律（置信度 0.3-0.7） | `$MEMORY_ROOT/AGENTS/GL2-模式候选.md` | 按需读取 |
| GL2b 项目规则 | 特定项目积累的规律 | `{项目}/.memory/.rules/` | 进入项目时读取 |
| GL2c 临时约定 | 当前对话观察 | 当前会话（不持久） | 对话结束即过期 |
