# 01 — 架构定义

> 定义五层记忆系统的核心结构：每一层的含义、存储位置、数据流向。

---

## 五层架构

| 层级 | 名称 | 内容 | 变化速度 | 存储位置 |
|------|------|------|---------|---------|
| L1 | 元信息层 | 价值观、底线、核心驱动力、金句 | 年/永久 | `$MEMORY_ROOT/核心层/CORE.md` |
| L2 | 认知模式 | 决策习惯、思维方式、交互偏好 | 季度 | `$MEMORY_ROOT/记忆规则/用户基本规则.md` |
| L3 | 行为规律 | 能量周期、工作节奏、习惯 | 月度 | `$MEMORY_ROOT/记忆规则/用户基本规则.md` |
| L4 | 情境状态 | 当前目标、卡点、项目进度 | 天/周 | `$WORKSPACE/.memory/MEMORY.md` |

### L1 元信息层

**定义**：你最底层的价值观、底线、核心驱动力。你"是什么样的人"的本质。

**特点**：
- 变化极慢，写一次管很久
- 必须有你确认才能写入或修改
- 不衰减，永久有效直到你主动改变

**来源**：
- 问卷 Q1-Q7
- L2 中持续 6 个月以上稳定的模式（AI 提案 + 你确认）

**文件**：`$MEMORY_ROOT/核心层/CORE.md`

### L2 认知模式层

**定义**：你习惯怎么思考、怎么做决策、怎么接收信息。

**特点**：
- 变化较慢（季度级）
- 置信度管理，自动写入但需佐证
- 6 个月未观察到开始衰减

**来源**：
- 问卷 Q8-Q17
- 从 L3 行为规律归纳（AI 自动 + 置信度评分）

**文件**：`$MEMORY_ROOT/记忆规则/用户基本规则.md`

### L3 行为规律层

**定义**：从项目级数据中提取的跨项目共性行为模式。

**特点**：
- 变化中等（月级）
- 从 L4 原始数据自动提取，不需要你确认
- 数据不出现后每周降置信度

**来源**：
- 多个项目的 L4 日记/进度数据泛化
- 对话中的行为观察

**文件**：`$MEMORY_ROOT/记忆规则/用户基本规则.md`（与 L2 同文件但分章节）

### L4 情境状态层

**定义**：你现在在做什么，当前项目的状态、目标和进度。

**特点**：
- 变化最快（天/周级）
- 按项目隔离，不进入公共区
- AI 自动记录，不需要提炼

**来源**：
- 日常对话
- AI 自动记录日记/进度

**文件**：`$WORKSPACE/.memory/MEMORY.md`

## 数据流向

### 提炼方向（从具体到本质）

```
L4 项目数据（日记/进度/任务）
  │ AI 在月度蒸馏时提取跨项目共性
  ▼
L3 行为规律（"截止日前你会冲刺"）
  │ 置信度 ≥ 0.7 持续 3 个周期 → 考虑晋升
  ▼
L2 认知模式（"你习惯先分析再行动"）
  │ 置信度 ≥ 0.9 持续 3 个月 → AI 提案
  ▼
L1 元信息（"结果比过程重要"）
  │ 你确认后锁定
```

### 预测方向（从本质到具体）

```
L1（"结果比过程重要"）
  │
L2（"截止日前会熬夜赶工"）
  │
L3（"最后 48 小时效率最高"）
  │
L4（"项目 A 还剩 3 天截止"）
```

## 系统常量

### 公共文件夹路径（多端共享）

| 常量名 | 路径 | 说明 |
|--------|------|------|
| `PATH_SHARED_ROOT` | `$MEMORY_ROOT` | 公共文件夹根目录 |
| `PATH_CORE` | `$MEMORY_ROOT/核心层/CORE.md` | L1 核心层 |
| `PATH_RULES_DIR` | `$MEMORY_ROOT/记忆规则/` | L2/L3 规则目录 |
| `PATH_RULES_USER` | `$MEMORY_ROOT/记忆规则/用户基本规则.md` | L2+L3 详细规则 |
| `PATH_RULES_CORE` | `$MEMORY_ROOT/rules/core-identity.md` | L1+L2 AI 参考规则（编译视图） |
| `PATH_RULES_BEHAVIOR` | `$MEMORY_ROOT/rules/behavior-rules.md` | L3 AI 参考规则 |
| `PATH_SKILLS_DIR` | `$MEMORY_ROOT/技能配置/` | Skill 模板源 |
| `PATH_GROWTH_BOX` | `$MEMORY_ROOT/成长箱/` | GL1 铁律 + GL2 模式候选 |
| `PATH_ROUTING_TABLE` | `$MEMORY_ROOT/rules/记忆路由表.md` | 信号词路由规则 |

### 工作区路径（项目级，按项目隔离）

| 常量名 | 路径 | 说明 |
|--------|------|------|
| `PATH_MEMORY_DIR` | `$WORKSPACE/.memory/` | 项目级工作记忆目录 |
| `PATH_MEMORY` | `$WORKSPACE/.memory/MEMORY.md` | L4 情境层载体 |
| `PATH_DIARY` | `$WORKSPACE/.memory/YYYY-MM-DD.md` | 每日日志 |

## AI 侧结构

| 组件 | 内容 | 存储位置 | 加载方式 |
|------|------|---------|---------|
| GL1 铁律 | 从你 L1 编译 + 跨平台通用规则 | `$MEMORY_ROOT/成长箱/01-铁律.md` | 每次对话必读 |
| GL2a 场景规则 | 跨项目合作规律（置信度 0.3-0.7） | `$MEMORY_ROOT/成长箱/02-模式候选.md` | 路由表触发 |
| GL2b 项目规则 | 特定项目积累的规律 | `{项目}/.workbuddy/.rules/` | 进入项目时读取 |
| AI 操作日志 | 每次对话的操作记录、成长箱审计 | 不持久 | 每次对话结束时写入 |
