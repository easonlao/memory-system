# 五层记忆系统 v5.0

## 安装

**三步完成：**

1. 打开 `AGENTS.md`，**复制全部内容**到你的 AI 平台入口文件（CLAUDE.md / .cursorrules 等）
2. 将 `技能配置/` 下的 5 个 Skill 安装到平台
3. 对 AI 说 **"安装记忆系统"**

## 后续

- `$MEMORY_DATA` 是唯一需要备份的目录，迁移时复制即可
- 日常使用：写日记、周报、月度校准 AI 会自动处理

## 文件说明

| 文件 | 用途 |
|------|------|
| `AGENTS.md` | AI 模板，复制到平台入口 |
| `技能配置/` | 5 个 Skill 文件，安装到平台 |
| `07-问卷.md` | 安装时引导填充的 33 道题 |
| `09-报告模板.md` | 安装完成后生成洞察报告 |
