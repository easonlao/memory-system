# 五层记忆系统 v5.0

## 安装

**只需两步：**

1. 将 `技能配置/` 下的 5 个 Skill 安装到平台
2. 对 AI 说 **"安装记忆系统"**

安装向导会自动配置平台入口文件、创建记忆数据目录、引导完成问卷。

## 后续

- `$MEMORY_DATA` 是唯一需要备份的目录，迁移时复制即可
- 日常使用 AI 会自动管理记忆，你只需要说"写日记"、"月度校准"等

## 文件说明

| 文件 | 用途 |
|------|------|
| `技能配置/` | 5 个 Skill，安装到平台 |
| `AGENTS.md` | AI 模板，安装时自动写入入口文件 |
| `07-问卷.md` | 安装时引导填充的 33 道题 |
| `09-报告模板.md` | 安装完成后生成洞察报告 |
