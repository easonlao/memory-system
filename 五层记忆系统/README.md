# 五层记忆系统 v5.0

一套让 AI 能持久理解你的记忆框架。数据存在你手里，迁移只需复制一个目录。

---

## 安装

### 1. 放置框架

把 `五层记忆系统/` 和 `技能配置/` 放到你的设备上（电脑、云盘均可）。

### 2. 配置平台入口

打开 `AGENTS.md`，将其中的内容**复制粘贴**到你的 AI 平台入口文件中：

| 平台 | 入口文件 |
|------|---------|
| Claude Code | `CLAUDE.md` |
| Cursor | `.cursorrules` |
| Gemini CLI | `GEMINI.md` |
| GitHub Copilot | `.github/copilot-instructions.md` |
| 其他 | 查阅对应平台文档 |

如果平台没有专门的入口文件，在第一条对话中粘贴也可以。

### 3. 安装技能

将 `技能配置/` 下的 5 个 Skill 文件安装到你的 AI 平台（具体方法见平台文档）。

---

## 首次使用

配置完成后，对 AI 说：

> **安装记忆系统**

AI 会引导你：
1. 选择记忆数据的存放位置（`$MEMORY_DATA`）
2. 回答 33 道问卷题，填充你的画像
3. 生成初始规则和报告

**之后就可以正常使用了。**

---

## 日常使用

记忆系统会在后台自动运行：

- 你写日记、做计划 → 记忆助理帮你记录
- 完成一个任务 → 自动写工作区日志
- 每月说"月度校准" → AI 从日记提炼你的行为模式
- AI 发现规律 → 自动优化合作方式

你不需要特意管理记忆文件，AI 会维护。

---

## 备份与迁移

`$MEMORY_DATA` 目录（默认 `~/.memory-data/`）是你**唯一需要备份**的内容。
换设备时，把这个目录复制到新设备，AI 就能无缝继续。
